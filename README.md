 
# 🏥 AI Medical Diagnosis Assistant

An AI-powered web application that predicts possible diseases based on the symptoms entered by the user. This project uses Machine Learning with Flask to provide disease predictions, descriptions, precautions, and health recommendations.

> **Note:** This project is developed for educational purposes only and is not intended to replace professional medical advice.

---

#  Features

- 👤 User Registration & Login
- 🤖 AI-Based Disease Prediction
- 🩺 Symptom-Based Diagnosis
- 📋 Disease Description
- 💊 Recommended Precautions
- 📜 Prediction History
- 💾 SQLite Database
- 📱 Responsive Web Interface
- 🌐 Ready for Cloud Deployment

#  Installation

## Clone Repository

```bash
git clone https://github.com/giddaiah-jogu/AI-Medical-Diagnosis-Assistant.git
```

## Open Project Folder

```bash
cd AI-Medical-Diagnosis-Assistant
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

#  Machine Learning

The project uses a **Decision Tree Classifier** trained on a disease symptoms dataset to predict the most likely disease based on selected symptoms.

Libraries used:

- Scikit-learn
- Pandas
- NumPy
- Joblib
